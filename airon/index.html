風憩セコロ × AI

System

AIRONは、風憩セコロの業務にAIを組み込むための自社システム。

単体の便利機能ではなく、写真、提案、設計、現場、ナレッジの各環境へ同じ思想で AI を広げていくための名前です。

その最初の入口として、画像業務を担う AIRON Image を立ち上げました。

PHOTO / PROPOSAL / DESIGN / FIELD / KNOWLEDGE

First Category

最初に立ち上がったのは AIRON Image。

AIRON の第一カテゴリーが AIRON Image です。その中で、すでに動いている最初のモジュールが AIRON Image Edit。次に AIRON Image Compose を加え、画像業務の入口を一つのカテゴリーとして育てます。

Live Module

AIRON Image Edit

日本語の指示で画像編集を行う、AIRON 最初の実装です。

Next Module

AIRON Image Compose

製品情報と現場写真をつなぎ、画像合成へ広げる次の実装です。

AIRON Image Edit の詳細を見る

Expansion

AIRON Image から、会社を取り巻く環境へ広げる。

AIRON は一つの画面で完結しません。カテゴリーを増やしながら、各業務の環境に自然な形で AI を埋め込みます。

Now

写真業務

AIRON Image Edit が稼働中。まずは画像を扱う入口を先に整えます。

Next

提案業務

AIRON Image Compose を加え、設置イメージ生成と提案づくりへつなげます。

Future

設計・現場・ナレッジ

図面、履歴、問い合わせ、現場記録まで、別カテゴリーとして順に広げていきます。

Platform

広がっても、裏側はひとつ。

カテゴリーごとに見える画面が変わっても、AIRON の基盤は共通化します。

Control

モデル選定、プロンプト補正、品質調整を共通運用する。

Connect

写真、製品DB、文書、履歴を業務文脈に沿って接続する。

Embed

AI 専用ツールを増やさず、既存の業務環境へ組み込む。

Improve

利用ログと現場の反応を回し、会社に合うシステムへ育てる。

Roadmap

AIRONはこの順で広げる。

まず一カテゴリーを実装し、次に厚みを出し、その後ほかの業務へ広げます。

01

AIRON Image を公開

AIRON Image Edit を先に実装し、画像業務の最初の入口をつくる。

02

AIRON Image Compose を追加

画像編集の次に、画像合成を加えてカテゴリーの厚みを出す。

03

別カテゴリーへ展開

AIRON Image で得た知見をもとに、提案、設計、現場、ナレッジへ順に広げる。

Philosophy

AIRONは、一つの機能名ではない。風憩セコロの仕事全体にAIを通すためのシステムである。

第一カテゴリー AIRON Image から始め、会社を取り巻くあらゆる環境へ広げていきます。
